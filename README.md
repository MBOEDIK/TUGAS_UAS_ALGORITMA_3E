## ğŸš€ Implementasi Algoritma

### 1. Huffman Coding
Algoritma kompresi data yang menggunakan prinsip *greedy* untuk menghasilkan kode biner dengan panjang minimum berdasarkan frekuensi karakter.
* **Proses**: Program menghitung frekuensi karakter, membangun pohon Huffman, dan menghasilkan kode biner unik untuk tiap karakter.
* **Statistik**: Menampilkan ukuran bit awal (ASCII) dibandingkan dengan bit hasil kompresi serta persentase efisiensi penghematan data.
* **Interaksi**: Pengguna memasukkan teks bebas pada area input untuk melihat hasil konversi binernya.

### 2. Algoritma Dijkstra
Algoritma untuk mencari jalur terpendek dari satu titik ke titik lainnya pada graf berbobot positif.
* **Graf**: Menggunakan struktur graf dengan 10 titik (node) dan 14 sisi (edge) yang saling terhubung dengan bobot tertentu.
* **Visualisasi**: Menggunakan HTML5 Canvas untuk menggambar graf secara dinamis.
* **Interaksi**: Pengguna menginputkan nomor titik awal (0-9), dan sistem akan mengalkulasi jarak terpendek ke seluruh titik lainnya dalam graf.
