Huffman Coding

Huffman Coding adalah algoritma kompresi data yang menggunakan prinsip greedy untuk menghasilkan kode biner dengan panjang minimum berdasarkan frekuensi karakter.

Algoritma Dijkstra

Algoritma Dijkstra digunakan untuk mencari jalur terpendek dari satu titik ke titik lainnya pada graf berbobot positif.
